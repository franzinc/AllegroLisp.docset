memuntrace

Arguments: &rest fnspecs

This macro removes the fwrappers created by memtrace for the specified functions. (See fwrappers-and-advice.htm.) Called with no arguments, this macro removes all memtrace fwrappers (analogously with (untrace) untracing all traced operators).

See Memlog: A Facility for Minimally Intrusive Monitoring of Complex Application Behavior in smp.htm for more information.

See multiprocessing.htm for general information on multiprocessing in Allegro CL.
