winspect

Arguments: object

UNIX only

This function displays an Allegro Composer Inspector window inspecting object.

Allegro Composer must already be running. It is an error to call this function when Allegro Composer is not running.

See composer.htm for general information on Allegro Composer. See The window-based inspector in that document for more information on Inspector windows.
