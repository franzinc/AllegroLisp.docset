scroll-window-into-parent

Arguments: window

If window is a child window, then calling this function will scroll its parent window as much as necessary for window to be fully visible in the interior of the parent window. If window is a top-level window, then nothing is done. For a discussion of parent and child windows, see see About child, parent, and owner windows in cgide.htm.
