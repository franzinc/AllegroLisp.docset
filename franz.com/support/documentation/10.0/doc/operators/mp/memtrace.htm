memtrace

Arguments: &rest fnspecs

This macro adds an fwrapper to each of the arguments (see fwrappers-and-advice.htm). This fwrapper makes two memlog calls:

  • The first call is on entry with the name of the function, a depth of recursion counter, and the arguments to this call.
  • The second call is on exit with the name of the function, the negative value of the recursion counter, and any values returned by the function.

See Memlog: A Facility for Minimally Intrusive Monitoring of Complex Application Behavior in smp.htm for more information.

See multiprocessing.htm for general information on multiprocessing in Allegro CL.
