presto-fasl-set

Arguments:

In previous releases, this function returned a list of filenames referenced by unresolved stub functions libfasl (Allegro Presto) loaded into the current image.In 7.0, the Allegro Presto facility has been removed. This function is kept for backward compatibility only. It always returns nil. See The Allegro Presto facility has been removed in loading.htm for further information.
