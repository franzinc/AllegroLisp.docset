mysql-insert-id

Arguments: db

This function is an accessor to a field of a mysql structure. db should be a mysql object returned by connect. The value of *mysql* is a suitable value. This function returns the value of the single autoincrement column in the last row inserted. This is a very powerful feature. See MySQL Examples in mysql.htm for more details.

See mysql.htm for information on the Allegro MySQL facility.
