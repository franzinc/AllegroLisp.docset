address-to-aligned

Arguments: n

Convert the integer n, which should be a pointer to an object in memory, to an aligned pointer. See aligned-to-address and Aligned Pointers in ftype.htm.

See ftype.htm for information on foreign types in Allegro CL and foreign-functions.htm for general information on foreign functions in Allegro CL.
