client-lisp

This class is used to represent each running client image in the controlling Lisp process. The appropriate instance is an argument to most control functions. Instances are created by run-other-client.

The accessor for this class are:

  • client-lisp-data
  • client-lisp-start
  • client-lisp-key (reader only)
  • client-lisp-keep (reader only)
  • client-lisp-port (reader only)
  • client-lisp-rc (reader only)
  • client-lisp-name (reader only)
  • client-lisp-errors
  • client-lisp-verbose

See Running several communicating Allegro CL images in rpc.htm for information on running several Lisp images.
